## [1.0.0] - 2026-05-24

### Added
- `envguard check` — validate .env against .env.example
- `envguard diff` — side-by-side key comparison between env files
- `envguard init` — generate .env.example from existing .env
- `envguard secrets` — detect leaked secrets and sensitive values
- Zero external dependencies
- Dual CJS/ESM output
- 51 tests
